# Numeralysis

## Overview
Numeralysis is a CLI tool written entirely in C, which is able to anaylse numbers or sets of data, and their corresponding relationships. Numeralysis can also graph data in various formats, predict data points using interpolation and extrapolation, and perform simple logical operations on datasets.

Numeralysis aims to be a fast and efficient CLI tool, without sacrificing accuracy.
